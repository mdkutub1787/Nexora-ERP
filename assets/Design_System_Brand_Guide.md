# Nexora ERP - Design System & Brand Guide

## Brand Overview
**Nexora ERP** is designed to be a smart, scalable, and seamless Enterprise Resource Planning system for modern businesses. The brand communicates trust, efficiency, and cutting-edge technology.

## 1. Brand Colors
The color palette uses a combination of corporate blue for trust and vibrant cyan for modern tech appeal.

- **Primary Azure Blue:** `#0D47A1` - Used for primary actions, app bar, and branding.
- **Vibrant Cyan:** `#00E5FF` - Used for highlights, accents, and secondary actions.
- **Dark Surface:** `#121212` - Used for dark mode backgrounds to reduce eye strain.
- **Light Surface:** `#F5F7FA` - Used for light mode backgrounds for a clean look.
- **Success Green:** `#00C853` - Used for positive metrics and successful actions.
- **Error Red:** `#D50000` - Used for warnings, low stock alerts, and errors.

## 2. Typography
The app uses clear, highly legible typography suitable for data-dense dashboards.
- **Headlines:** Bold, commanding, used for Screen Titles and large metrics.
- **Body Text:** Regular weight, clean sans-serif for lists, forms, and general content.

## 3. Logo & Iconography
- **App Icon:** A geometric logo that symbolizes integration and data flow. Designed to stand out on mobile home screens.
- **Icons:** Material Icons (Rounded) are used throughout the app for consistency.

## 4. UI Components (Material 3)
- **Cards:** Used extensively on the Dashboard and Product List. Elevated with subtle shadows.
- **Buttons:** Fully rounded corners for primary buttons to give a friendly, clickable feel.
- **Text Fields:** Outlined text fields with smooth corner radius.
